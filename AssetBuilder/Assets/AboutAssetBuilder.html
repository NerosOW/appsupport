Asset Builder

Asset Builder, add and manage assets for including in your Shortcut.

When finished adding pictures, video, audio files/text Asset Builder shares a json/dictiionary that you paste into a text box in your Shortcut. You then use the asset anywhere in your Shortcut with a dictionary variable which you can use in lots of action boxes.

You can just share back to Asset Builder if you need to add/delete or copy assets.

You can easily reuse your dictionaries in other Shortcuts
Easier than transferring several long strings into a Shortcut dictionary
Individual values can be viewed/decoded, deleted or shared
Zip several files together

Asset Types

Base 64 encoded

  • Zip Files
  • Images
  • Audio

Text

  • vcards
  • Dictionaries
  • HTML

Getting Started

Asset Builder will ask if you want to load an already created piece of json from clipboard or file.

The Main Menu

The main menu consists of commands for adding assets, a command to share your finished json, and a display of the items you've already added.

Photo Library

Select a file from the Photo Library to add.

File

Select any files you want to add, any images and media will show you the size of the item and offer the chance to change/resize/encode.

Zip

Here you can select several files then use open to zipped, encode and add to the assets as a single item.

Clipboard

You can add an asset from the Clipboard

Share

When you’ve finished you can go here to share the json produced by Asset Builder

Assets

The assets you have loaded, and named are shown at the bottom of the menu. Selecting the Assets list item will show you the assembled json, selecting any of the individual assets will let you view the string, decode, or delete them from your assets.
